Om Mig

Erfaren mjukvaruutvecklare med expertis inom inbyggda system, webb- och applikationsutveckling. Starkt fokus på användargränssnitt och UX-design samt erfarenhet av CI/CD och Git för att effektivisera utvecklingsprocessen. Färdigheter • Mjukvaruutveckling: Erfarenhet av inbyggda system, webb och applikationsutveckling. Expertis inom frontend-tekniker (JavaScript, HTML, CSS) och backend-tekniker (C/C++, C#, Python). • Användargränssnitt och UX-design: Passion för att skapa positiva användarupplevelser genom välutformade gränssnitt. • CI/CD och Git: Intresse och erfarenhet av kontinuerlig integrering/kontinuerlig distribution och användning av Git för att effektivisera utvecklingsprocessen..
